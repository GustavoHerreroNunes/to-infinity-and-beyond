Sobre

Este game foi desenvolvido como uma atividade avaliativa pelo aluno Gustavo Herrero Nunes, na turma de graduação (bacharelado) à distância em Sistemas de Informação, na FIAP (Faculdade de Informática e Administração Paulista) - Unidade Aclimação.

O objetivo era praticar as habilidades em Javascript vanilla (sem frameworks ou libs) para o desenvolvimento de um canvas-game de uma nave espacial desviando de asteroides. O jogo foi entregue como apresentado nesta página, porém um layout com bootstrap foi criado depois para que o projeto fosse adicionado ao portfólio.
